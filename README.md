# Blockchain Certificate Verification DApp

## Overview
This project is a decentralized application (DApp) that verifies certificates using blockchain technology.

## Features
- Secure certificate storage using smart contracts
- SHA-256 hashing for data integrity
- QR code-based verification
- Role-based access (admin control)
- Interactive UI with real-time feedback

## Tech Stack
- Solidity (Smart Contracts)
- Hardhat
- Ethers.js
- MetaMask
- HTML, CSS, JavaScript

## How it Works
1. Admin adds certificate
2. Data is hashed and stored on blockchain
3. QR code is generated
4. Users verify certificate using ID or QR
